# Interactive Personal Data Collector

## About This Project

This is a simple Python project that collects personal information from the user and displays it.

I made this project to practice basic Python concepts like taking input, using variables, data types, and type conversion.

## Features

* Takes user's name as input
* Takes user's age
* Takes user's height
* Takes user's favorite number
* Displays collected information
* Shows data type of each value
* Shows memory address using `id()`
* Calculates user's birth year
* Converts height from float to integer

## Python Concepts Used

* `input()` function
* Variables
* Data Types (`str`, `int`, `float`)
* Type Conversion
* `type()` function
* `id()` function
* f-string formatting

## How to Run

Make sure Python is installed on your computer.

Run the program using:

```bash
python main.py
```
